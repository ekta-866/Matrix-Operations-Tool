# Matrix Operations Tool

## Description
This project is a Matrix Operations Tool developed using Python and NumPy. It allows users to perform various matrix operations through a simple command-line interface.

## Features
- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Matrix Determinant
- Error Handling for Invalid Inputs

## Technologies Used
- Python 3
- NumPy

## How to Run
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run:

```bash
python matrix_operations.py
```

4. Enter the matrix dimensions and values.
5. Select the desired operation from the menu.
